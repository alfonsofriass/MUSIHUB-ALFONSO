# MusiHub

Base de implementación del TFG `MusiHub`.

## Estructura
- `front/`: futura app móvil Flutter.
- `back/`: backend mínimo en `FastAPI`.
- `MUSIHUB_PROJECT_CONTEXT.md`: contexto funcional y decisiones vivas del proyecto.
- `db-diagram.txt`: referencia del esquema de base de datos.

## Estado actual
- El backend se ha reducido a la mínima base útil para aprender `FastAPI` paso a paso.
- La base de datos y las migraciones se reintroducirán más adelante, cuando toque.

## Alcance de la base actual
- Aplicación `FastAPI` mínima.
- Dependencias mínimas para ejecutarla en local.
